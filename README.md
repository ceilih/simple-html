# simple-html

A mock clothing webpage made using html, css, and a little bit of JS. 

Hosted on Netlify at https://steady-alpaca-3f7b31.netlify.app/

All links are placeholders except for the ones shown in the modal window.
The form on the secondary page (reachable through the link on the modal window) is responsive and uses validation.
